# Raspberry PI

Der Raspberry PI ist ein universeller Einplatinencomputer.
Der einfach aufgebaute und günstige Rechner kann für eine
Vielzahl an Aufgaben (oder einfach nur zum Experimentieren)
verwendet werden.
