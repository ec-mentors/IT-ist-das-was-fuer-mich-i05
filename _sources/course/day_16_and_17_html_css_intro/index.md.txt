# Tag 16-17: Webseiten mit HTML und CSS

## HTML & CSS

Man kann Webseiten in Markdown schreiben, und sie von einem Programm wie [Sphinx](https://www.sphinx-doc.org/en/master/) übersetzen lassen - so wie unsere Kursseite. Oder man schreibt sie direkt in HTML, und verwendet CSS für die Formatierung. Es gibt auch Baukasten-Systeme, mit denen man ohne Programmierkenntnisse Webseiten bauen kann.

Hier findest du eine kleine [HTML-Datei](../../_static/mini_index.html) und eine [CSS-Datei](../../_static/mini_style.css). Speicher sie auf deinem Laptop (beide im gleichen Ordner), und erweitere sie nach Lust und Laune! Dein Code in der HTML-Datei kommt dort hin, wo ```<!--page content-->``` steht.
