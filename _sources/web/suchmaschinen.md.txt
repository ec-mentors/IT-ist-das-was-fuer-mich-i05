# Suchmaschinen
## Was ist eine Suchmaschine?
Eine Suchmaschine ist eine Website, über welche User nach Inhalten im Internet suchen können.

## Welche Suchmaschinen gibt es?
- Google
- Bing
- Yahoo!
- DuckDuckGo
- YANDEX

## Wie funktioniert eine Suchmaschine?

Die Suchmaschine ist ein Programm, das im Web-Browser nach Inhalten sucht. Gibst Du z.B. Dein Fokus Keyword in das Suchfeld ein, präsentiert Dir das Programm innerhalb weniger Sekunden die Ergebnisse. Diese sind auf Seiten enthalten, welche nach einem bestimmten Algorithmus sortiert sind.

## Arbeitsweise :

Crawler (auch Spider oder Bots genannt) 'krabbeln' über Websites und sammeln Daten.
Der Index ist die Auflistung von Webseiten, die auf eine Suchanfrage eines Nutzers hin von der Suchmaschine ausgegeben werden.
Suchmaschinen-Ranking bezeichnet die Reihenfolge, in der Suchergebnisse in den Search Engine Result Pages (SERPs) angezeigt werden.

Und die sind die 3 Schritt-Methoden von Google
1) Datensammlung - Crawling
2) Datenanalyse - Index
3) Datenausgabe - Ranking

## Datenschutz und Privatsphäre

Grundsätzlich ist eine Online-Suchmaschine auf die Daten ihrer Nutzer angewiesen. Je mehr und umfangreicher diese sind, desto präziser sind die Suchalgorithmen und so letzten Endes auch die Suchergebnisse.
