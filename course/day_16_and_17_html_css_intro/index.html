Tag 16-17: Webseiten mit HTML und CSS#

HTML & CSS#

Man kann Webseiten in Markdown schreiben, und sie von einem Programm wie Sphinx übersetzen lassen - so wie unsere Kursseite. Oder man schreibt sie direkt in HTML, und verwendet CSS für die Formatierung. Es gibt auch Baukasten-Systeme, mit denen man ohne Programmierkenntnisse Webseiten bauen kann.

Hier findest du eine kleine HTML-Datei und eine CSS-Datei. Speicher sie auf deinem Laptop (beide im gleichen Ordner), und erweitere sie nach Lust und Laune! Dein Code in der HTML-Datei kommt dort hin, wo <!--page content--> steht.

mini_index.html

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <title>title</title>
    <link rel="stylesheet" href="mini_style.css">
    <script src="script.js"></script>
</head>

<body>
    <h1>Header Text in h1 Tags</h1>
    <!-- page content -->
</body>

</html>

mini_style.css

body {
    background-color: lightblue;
}

h1 {
    color: white;
    text-align: center;
}

p {
    font-family: verdana;
    font-size: 20px;
}
